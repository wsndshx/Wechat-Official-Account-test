# Wechat-Official-Account-test
用Go编写的微信公众号测试后端

# 用法

### Windows

```bash
git clone https://github.com/wsndshx/Wechat-Official-Account-test.git
cd /Wechat-Official-Account-test
go build
```

### Linux

```bash
git clone https://github.com/wsndshx/Wechat-Official-Account-test.git
cd /Wechat-Official-Account-test
chmod a+x build.sh
./build.sh
```
